What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage.

With CSS, you can control, font, the size of text, the spacing between elements, how elements are positioned an laid out, what background images or background colors are to used, different display for defferent devices and sceens sizes, and much more!

Tip: The word cascading means that a style applied to a parent elment will also apply to a children elements within the parent.So, if you set the color of the body text to blue, all headings, paragraphs, and other text element within the body will also get the same color(unless you specify something else)!
